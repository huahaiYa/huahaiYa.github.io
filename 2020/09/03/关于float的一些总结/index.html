关于float的一些总结

Hello，这是我的第一篇博文哦，今天主要想记录关于浮点类型在内存中是怎么表现的，下面开始吧

float类型

浮点类型是c语言中的基本类型之一，float分为了有符号的和无符号的，在计算机的内存中没有符号之分，但是有正负之分，对于float类型来说正数和负数在内存中是不同的表现形式，下面就来看看吧！

flaot类型的表现形式

float类型在计算机中的存储方式：

31 ~30 30~22 22~0
符号位（1位） 指数部分（8位） 尾数部分（23位）

可以看到在内存中float类型被分为了三部分进行计算，那么他的值是怎样进行填写的，主要由以下几个步骤

1.将这种实数的绝对值转化为二进制

2.将二进制形式的小数点左移或者右移n位，直到小数点移动到第一个有效数字的右边

3.从小数点右边第一位开始输出二十三位放入第22位到0位

4.如果实数是正的，则在第31位放入0，否则放入1

5.如果n是左移得到的，说明指数是正的，那么第三十位放入1，如果n是右移得到的或者n=0，则第三十位放入0

6、如果n是左移得到的，则将n减去1后化为二进制，并在左边加“0”补足七位，放入第29到第23位。 如果n是右移得到的或n=0，则将n化为二进制后在左边加“0”补足七位，再各位求反，再放入第29到第23位。

下面通过三个实例来说明一些转化过程

8.25转化为浮点存储

首先将实数部分进行转化：对8求余数进行转化二进制。

求余 余数

8/2 = 4 0

4/2 = 2 0

2/2 = 1 0

1/2 = 0 1

结果逆序为二进制的值，既 1000

小数部分转化为二进制的值，对2进行乘法得整数位

0.25 * 2 = 0.5 0

0.5*2 = 1.0 1

小数部分正序得到结果，最终转化为二进制： 1000.01

最后对结果进行存储

首先对小数点进行左移直到遇到第一个非0数既 1.00001 * 2的三次方，小数点左移了3位，指数为3 （第二步）

（3）将小数位放到第23到0的位置，不足的补0；既00001000000000000000000

（4）8为正数，所以第31位为0

（5）前面已知小数位左移三位，所以将1放到第30位，并且将3-1=2的二进制放到29到23位 ，2的二进制为0000010

所以最终的结果为

0 10000010 00001000000000000000000

转化为16进制为：41040000
